📖 O que cada aba faz
🔒
Ofuscar
Transforma código Lua num formato ilegível com até 5 camadas de proteção (XOR, Shift, Base64, Reverse, pcall). Use antes de distribuir seu script FiveM.
🔓
Deofuscar
Recupera o código original a partir de um código ofuscado por este programa. Requer as mesmas chaves XOR e Shift usadas na ofuscação.
⚡
Cruzar
Analisa dois arquivos juntos e encontra eventos em comum, mesmo que um esteja ofuscado. Modo normal (back aberto) e reverso (front aberto).
🔑
Brute Force
Testa até 5.100 combinações de chaves para descobrir a que foi usada. Funciona quando você sabe pelo menos uma palavra do código original.
🗝️ O que são XOR e Shift?
São os segredos da ofuscação.
XOR (1–255) — embaralha cada byte com operação XOR bitwise.
Shift (1–20) — desloca cada caractere N posições no código ASCII.
Pipeline: código → Base64 → Reverse → Shift → XOR → array de bytes.
⚠️ Guarde sempre as chaves. Sem elas, só Brute Force pode recuperar.
🎮 Passo a passo FiveM
1
Abra 🔒 Ofuscar, cole seu client.lua e clique em Ofuscar
2
Anote as chaves XOR e Shift exibidas — necessárias para deofuscar depois
3
Baixe o .lua protegido e substitua o original na pasta de resource
4
Para analisar scripts alheios com back aberto + front ofuscado: use ⚡ Cruzar
5
Se não souber as chaves: use 🔑 Brute Force com um evento que você sabe que existe
🔒 Tudo roda localmente no seu navegador. Nenhum código é enviado para servidores externos. Código-fonte aberto e auditável.
